GPTE Reporting 2 Way TLS Authentication

Please execute the following to be able to participate in 2-Way TLS authentication with the GPTE Reporting RESTful API:


1. Create new directory to store the GPTE client key:
sudo mkdir -p /etc/gpte/certs

2. Copy certs from prod to client filesystem:
sudo scp <your gpte opentlc userId>@reporting.opentlc.com:/etc/gpte/certs/prod*   /etc/gpte/certs

3. test GPTE Reporting RESTful API using certs:
curl -v -k \  --cert /etc/gpte/certs/prod_gpteclient.pem \
 --key /etc/gpte/certs/prod_gpteclient.key \
 -X GET https://reporting.opentlc.com/gpte-reporting/rest/sanityCheck/
